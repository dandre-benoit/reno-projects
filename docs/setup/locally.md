# Renovation Project Management API

## Run locally

### Prerequises

- File .env created and configured (see .env.local file)
- Node.js installed 
- NPM installed
- PostgreSQL installed and configured (see .env.local file)
- [See technologies](../technologies/index.md)

### Endpoint
```bash
{PROTOCOL}://{HOST-NAME}:{PORT}/api

# Default configuration
http://localhost:3000/api
```

### Local Endpoint
http://localhost:3000/api

### Project Setup

```bash
$ npm install
```

### Compile And Run The Project

```bash
# development
$ npm run dev

# watch mode
$ npm run watch

# build file for production mode
$ npm run build

# production mode
$ npm run prod
```

### Run Tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
